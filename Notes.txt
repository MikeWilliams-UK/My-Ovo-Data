Instructions, download all files to a single folder.

You can then fire up the exe using the following command line to run it

OvoData.exe -u your.login@yourdomain.co.uk -p yourPassword -f 01/01/2020 -t 04/02/2020 -o S:\Data\OvoData.csv

-u  Your OVO username that you use to log in to the web site or mobile app with.
-p  Your OVO password that you use to log in to the web site or mobile app with.
-o  The path and filename to write the results to
-f  The first date to retrieve 1/2 hourly data for
-t  The (inclusive) last date to retrieve 1/2 hourly data for


Do NOT Put dates in the future.

Enjoy!

For replicating this in other languages, the process seems to be as follows:

Login by sending a POST request to https://my.ovoenergy.com/api/v2/auth/login with the following JSON in the body:
{
    "username": "...",
    "password": "...",
    "rememberMe": true
}

From the response, save the cookies to use in all future requests.
Next, send a GET request to https://smartpaym.ovoenergy.com/api/customer-and-account-ids . Be sure to save the items in the "accountIds" property (usually only one item).
Finally, make a GET request to https://smartpaym.ovoenergy.com/api/energy-usage/daily/{account_id}?date={yyyy-MM} where {account_id} is the value from the previous request and {yyyy-MM} is the month in the format 2020-08.

You can also get monthly or half-hourly usage using the following URLs similar to step 3:

https://smartpaym.ovoenergy.com/api/energy-usage/half-hourly/{account_id}?date={yyyy-MM-dd}
https://smartpaym.ovoenergy.com/api/energy-usage/monthly/{account_id}?date={yyyy}
